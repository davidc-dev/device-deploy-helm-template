device-app Helm Repository

This GitHub Pages site hosts the packaged device-app Helm chart.

  1. Add the repo:
helm repo add device-app https://davidc-dev.github.io/device-deploy-helm-template/
helm repo update
  1. Install the chart:
helm install my-device device-app/device-app

The Helm client reads index.yaml from this site to discover available chart versions.
